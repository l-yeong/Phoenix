
## 주제

Spring Boot,React 기반 “야구 예매 사이트” 웹 구현

## 간단한소개


 일반·시니어 모드와 음성 안내 기능을 통해 고령층의 접근성을 보완하고 알고리즘 기반 자동매칭과 시니어 전용 좌석 우선 배정을 제공하고
좌석 교환 기능으로 재예매의 불편을 줄여 효율적인 예매 경험을 지원합니다.

## 구현기능 / 팀원별


#### 민성호(조장) : 
- 프로젝트 일정 관리
